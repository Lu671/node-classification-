# node-classification
classify GitHub developers nodes based on two different GNN 
models, GCNConv and GATConv.

# Dataset:
https://snap.stanford.edu/data/github-social.html

# Results:
test accuracy of GCN model 87%
test accuracy for GAT model is 83%
